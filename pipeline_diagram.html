모듈 2

NLP 리뷰 기반 키워드 · 순위권 검색어

① 형태소 분석 (Okt POS)
ReviewTfidfAnalyzer.extract_per_review(reviews)
dict { review_id: Counter(keyword→count) }
고기맛있다 데이트분위기 접근성회식
② 블랙리스트 제거 + 표현 통일
KeywordNormalizer.normalize(counter)
Counter (정제 후)
맛있다 좋다 제거 → 고기구이 (표현 통일)
③ TF-IDF 가중치 산출
ReviewTfidfAnalyzer.compute_tfidf(per_review)
dict { keyword: tfidf_score }
이 매장에서만 자주 등장하는 차별화 키워드를 높은 점수로 식별
④ 순위권 검색어 (크롤링 데이터)
merge_keywords(place_id, nlp_tfidf, nlp_per_review)
tuple (merged_tfidf, merged_per_review, keyword_meta)
해당 검색어로 검색했을 때 우리 매장이 70위권 내 노출되는 검색어를 NLP 결과에 통합. 랭크 정보(rank_no, rank_no_change)를 keyword_meta에 보존.
순위권 검색어 조건
· 크롤링 데이터 기반 (rankings 테이블)
· 해당 키워드 검색 시 우리 매장 노출 순위 ≤ 70위
· rank_no, rank_no_change 정보 존재
필드설명예시
rank_no현재 노출 순위23위
rank_no_change순위 변동폭+5 (상승)
📤 최종 출력 예시
강남역 고기집 강남역 데이트 강남 회식장소 혼밥 가능
↑ NLP 키워드 + 70위권 순위권 검색어 통합 결과
